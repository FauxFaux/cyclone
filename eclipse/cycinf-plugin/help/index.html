Cyclone with Inference Plugin

The Cyclone with Inference Plugin ("cycinf-plugin") is a tool for use with the Cyclone with Inference compiler, which must be installed separately. The plugin loads the inference decisions the compiler made on your code and presents them in several Eclipse views so you can see what happened and improve your program.

This brief guide to using the plugin consists of the following pages:

  • Setting up and building a project
  • Loading inference data
  • Browsing inference data

More information about Cyclone may be found at the Cyclone Web site, http://www.eecs.harvard.edu/~greg/cyclone/.
