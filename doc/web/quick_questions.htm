Quick Questions About Cyclone

Version 1.0

We have designed Cyclone so that, "if it looks like C, it acts like C."  However:

  • Because Cyclone is a safe language, the compiler rejects some C programs.  Because the compiler must err conservatively, it rejects some safe C programs.

  • Cyclone has many additions to C to make programming more convenient and to express stronger invariants than you can in C.

Here we imagine an experienced C programmer is reading or developing Cyclone code and is frustrated by strange syntax or cryptic error messages.  We keep the explanations short so you can get back to hacking and look up a longer explanation in the user's manual later.
